# React + Kakao Map

api doc: https://apis.map.kakao.com/web/guide/

## Requirements

### 1. Kakao 지도 JavaScript API는 키 발급을 받아야 사용할 수 있다

### 2. Kakao account가 필요하다. (한국인은 이미 있을것이라고 생각)

### 3. [카카오 개발자 사이트](https://developers.kakao.com)로 이동
